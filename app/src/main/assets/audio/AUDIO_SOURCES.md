## Download Sources

### Free Sound Effects (Recommended)

1. **Mixkit** (https://mixkit.co/free-sound-effects/)
   - License: Free, no attribution required
   - Format: MP3
   - Quality: High

2. **Pixabay** (https://pixabay.com/music/sound-effects/)
   - License: Free, no attribution required
   - Format: MP3
   - Quality: Good

3. **Freesound.org** (https://freesound.org/)
   - License: Varies (CC0 preferred, CC-BY acceptable)
   - Format: WAV/MP3/OGG
   - Quality: High

4. Update this file with source and license information
5. Check off "Status" when file is added
