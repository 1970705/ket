### Epic #3: 动态星级评分 (2-3人天)

**负责人**: android-engineer + game-designer

#### Story #3.1: 评分算法实现 (1天)
- [ ] 算法实现 (准确率 + 思考时间 + 提示 + 错误 + 连击)
- [ ] 单元测试 (覆盖率 > 80%)
- [ ] 边界情况测试
- [ ] 性能测试

**评分算法**:
```kotlin
fun calculateStars(
    correctCount: Int,
    totalCount: Int,
    avgTimeTaken: Long,
    minTimeThreshold: Long,
    hintsUsed: Int,
    maxCombo: Int,
    errorCount: Int
): Int {
    // 准确率 (0.0 - 1.0)
    val accuracy = correctCount.toFloat() / totalCount.toFloat()

    // 思考时间系数 (盲目快速点击惩罚)
    val timeCoefficient = when {
        avgTimeTaken < minTimeThreshold -> 0.7f  // 太快
        avgTimeTaken > minTimeThreshold * 3 -> 0.8f // 太慢
        else -> 1.0f
    }

    // 提示惩罚 (使用提示最多2星)
    val hintPenalty = if (hintsUsed > 0) -1 else 0

    // 连击奖励 (高连击可以抵消错误)
    val comboBonus = when {
        maxCombo >= 10 -> 1
        maxCombo >= 5 -> 0
        else -> 0
    }

    // 基础星级
    val baseStars = when {
        accuracy >= 0.9f -> 3
        accuracy >= 0.7f -> 2
        accuracy >= 0.5f -> 1
        else -> 0
    }

    // 最终星级 (考虑各种因素)
    val finalStars = baseStars + hintPenalty + comboBonus

    return finalStars.coerceIn(0, 3)
}
```

**验收标准**:
- ✅ 算法正确计算星级
- ✅ 单元测试覆盖率 > 80%
- ✅ 边界情况处理正确
- ✅ 性能测试通过 (< 10ms)

**估计**: 1天
