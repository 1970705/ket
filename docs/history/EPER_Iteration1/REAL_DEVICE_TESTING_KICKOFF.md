## 📋 快速参考

### ADB命令

```bash
# 检查设备
adb devices -l

# 安装应用
adb -s 5369b23a install -r app/build/outputs/apk/debug/app-debug.apk

# 启动应用
adb -s 5369b23a shell am start -n com.wordland/.ui.MainActivity

# 查看日志
adb -s 5369b23a logcat | grep Wordland

# 截图
adb -s 5369b23a shell screencap -p /sdcard/screenshot.png
adb -s 5369b23a pull /sdcard/screenshot.png

# 录屏
adb -s 5369b23a shell screenrecord /sdcard/demo.mp4
adb -s 5369b23a pull /sdcard/demo.mp4
```
