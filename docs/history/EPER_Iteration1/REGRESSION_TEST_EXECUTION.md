## 🎯 回归测试范围

### Phase 1: 环境准备 (5 分钟)

**前置条件检查**:
```bash
# 1. 检查设备连接
adb devices -l

# 预期输出:
# 24031PN0DC    auroropro (API 36)

# 2. 清除应用数据（确保干净状态）
adb shell pm clear com.wordland

# 3. 重新安装最新版本
./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### Phase 2: 核心场景回归测试 (20 分钟)

**测试场景清单**:

#### TC-REG-001: 应用启动和导航

**步骤**:
1. 启动应用: `adb shell am start -n com.wordland/.ui.MainActivity`
2. 观察启动速度
3. 验证主界面四个选项显示
4. 截图证据

**验收标准**:
- ✅ 应用正常启动，无崩溃
- ✅ 主界面显示正确
- ✅ 四个选项可点击

---

#### TC-REG-002: 完整学习流程 (关键测试)

**步骤**:
1. 从主界面选择 "Start Learning"
2. 选择 "Look Peninsula" → "Level 1"
3. 完成完整 6 个单词学习
4. 观察每个单词的:
   - 字母飞入动画
   - 答案反馈（"Correct!"）
   - 连击提示（"Combo x2", "Combo x6"）
5. 完成后观察星级显示
6. 截图证据（至少 3 张）

**验收标准**:
- ✅ 所有 6 个单词正常显示
- ✅ 字母飞入动画流畅
- ✅ "Correct!" 反馈清晰
- ✅ 连击系统工作正常
- ✅ 星级显示正确 ⭐⭐⭐
- ✅ 无崩溃或异常

---

#### TC-REG-003: 地图导航和关卡选择

**步骤**:
1. 从主界面选择 "Explore Islands"
2. 观察 "Look Peninsula" 显示
3. 点击进入关卡选择
4. 观察 5 个关卡显示
5. 返回主界面
6. 截图证据

**验收标准**:
- ✅ 岛屿地图显示正确
- ✅ 关卡列表显示正确
- ✅ 导航流畅无迷失
- ✅ 返回功能正常

---

#### TC-REG-004: 进度保存验证

**步骤**:
1. 完成 Level 1 后关闭应用
2. 重新启动应用
3. 进入关卡选择
4. 验证 Level 1 显示为已完成

**验收标准**:
- ✅ 进度正确保存
- ✅ 星级记录正确
- ✅ Level 2 已解锁

---

### Phase 3: 性能验证观察 (10 分钟)

**性能观察任务**:

#### TC-PERF-001: 内存使用

**步骤**:
1. 打开 Android Studio Profiler
2. 连接到 auroropro 设备
3. 观察 com.wordland 进程
4. 记录以下数据:
   - Native Heap
   - Dalvik Heap
   - TOTAL PSS
5. 对比基线: 176 MB PSS

**验收标准**:
- ✅ TOTAL PSS < 300MB
- ✅ 无明显内存泄漏
- ✅ 性能无明显退化

---

#### TC-PERF-002: 动画流畅度

**步骤**:
1. 执行完整学习流程
2. 观察字母飞入动画
3. 观察连击和星级动画
4. 记录流畅度

**验收标准**:
- ✅ 动画流畅无卡顿
- ✅ 无明显掉帧
- ✅ 视觉体验优秀

---

#### TC-PERF-003: 响应时间

**步骤**:
1. 测试按钮点击响应
2. 测试界面切换速度
3. 测试返回导航

**验收标准**:
- ✅ 按钮响应迅速
- ✅ 界面切换流畅
- ✅ 无明显延迟
