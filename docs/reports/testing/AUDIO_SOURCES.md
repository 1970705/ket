## License Codes

- **CC0**: Creative Commons Zero - Public Domain, no attribution required
- **CC-BY**: Creative Commons Attribution - Attribution required
- **Free (no attribution)**: Free to use without attribution
- **Fair Use**: Educational fair use of dictionary pronunciations

## Notes

1. Always verify the license before using any audio file
2. Keep attribution notices in the app's About/Credits section if required
3. For CC-BY licensed content, add attribution to:
   - Google Play Store description
   - In-app Settings > Credits section
   - README.md file
