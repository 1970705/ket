## 🧪 测试方法

### 集成测试策略

由于这些是 Compose UI 组件，测试策略为：

1. **配置验证**: 直接测试常量和配置值
2. **状态逻辑**: 验证可见性、解锁条件
3. **动画参数**: 验证时长、缓动、偏移
4. **UI规范**: 验证尺寸、间距、颜色

**不测试** (需要 Compose UI 测试):
- 实际渲染效果
- 动画执行过程
- 用户交互
