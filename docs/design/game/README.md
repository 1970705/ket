# 游戏设计

本目录包含 Wordland 的游戏设计文档，包括游戏机制、用户体验、奖励系统等核心设计。

## 🎮 游戏概述

**Wordland** 是一款面向 6-12 岁儿童的 KET/PET 词汇学习游戏，通过游戏化机制让英语学习变得有趣和高效。

### 核心设计原则
1. **游戏为先** - 游戏性优先，教育内容融入游戏玩法
2. **即时反馈** - 每个操作都有明确和及时的反馈
3. **小步快跑** - 任务分解为小目标，持续成就感
4. **儿童友好** - 适合 10 岁孩子的认知水平
5. **防沉迷** - 平衡娱乐性和教育性，防止过度沉迷

## 🎯 当前游戏模式

### 1. Spell Battle（拼写战斗）
- **状态**: ✅ 完全功能
- **描述**: 看中文翻译拼写英文单词
- **玩法**: 虚拟键盘 + 即时反馈
- **进度**: 30 个 KET 词汇，5 个关卡

### 2. 未来游戏模式（规划中）
- Listen Find（听音辨词）
- Sentence Match（句子配对）
- Quick Judge（快速判断）

## 🎨 游戏元素

### 关卡结构
- **岛屿**: 主题区域（如 Look Island）
- **关卡**: 每个岛屿 6 个单词
- **难度**: 1-5 级难度递进

### 奖励机制
- ⭐ **星级评分**: 1-3 星动态评分
- 🏆 **成就系统**: 多种成就类型
- 📈 **等级系统**: 角色成长和晋升
- 💎 **虚拟货币**: 金币和奖励

### 反馈系统
- ✅ 正确反馈: 动画 + 音效 + 星星
- ❌ 错误反馈: 提示 + 鼓励
- 💡 提示系统: 3 级渐进提示
- 📊 进度条: 实时进度显示

## 👥 目标用户

### 主要用户
- **年龄**: 6-12 岁
- **英语水平**: KET/PET 初级
- **学习目标**: 词汇量扩充

### 用户特征
- 注意力持续时间短（10-15 分钟）
- 需要即时反馈和奖励
- 喜欢探索和收集
- 容易受游戏化影响

## 🎯 游戏循环

### 核心循环
```
选择关卡 → 学习单词 → 拼写答题 → 获得反馈
→ 解锁成就 → 获得奖励 → 提升等级 → 选择下一关卡
```

### 学习循环
```
新词学习 → 初次测试 → 复习强化 → 长期记忆
```
