```
EMOTION
  Excitement │      ╱╲
             │     ╱  ╲    ╱╲         ╱────╲
             │    ╱    ╲  ╱  ╲       ╱      ╲
  Neutral   │───╱      ╲╱    ╲_____/        ╲___
             │0   1   2   3   4   5    10   20  30 (minutes)

Curiosity    ═══►╮               ╭──╮
Ownership        ╰──►╮           ╭──╯  ╰──╮

Achievement               ╰─╯
Anticipation                      ╰────►╮

KEY MOMENTS:
0:00 - App opens (Curiosity)
0:30 - Avatar selected (Ownership)
1:30 - Tutorial complete (Competence)
2:00 - First victory (Achievement)
2:30 - Map preview (Anticipation)
5:00 - Session 1 complete (Satisfaction + Anticipation)
Day 2 - Session 2 (Welcome back)
Day 2 - Level 1 complete (Milestone)
Day 3-7 - Session 3 (Challenge + Recognition)
```

## Critical User Behaviors to Track

| Event | When | Why Important |
|-------|------|---------------|
| `onboarding_started` | Cold open loads | Measure drop-off at entry |
| `avatar_selected` | User picks avatar | Personal investment |
| `tutorial_completed` | CAT spelled | Measure tutorial friction |
| `first_word_complete` | First real word | Initial success feeling |
| `session_1_complete` | 3 words done | Day 0 success |
| `session_2_start` | User returns | Day 1 retention |
| `level_1_complete` | First milestone | Progress validation |
| `achievement_unlocked` | First badge | Long-term motivation |
| `hint_used` | User taps hint | Measure difficulty |
| `avatar_tapped` | User interacts | Measure engagement |
