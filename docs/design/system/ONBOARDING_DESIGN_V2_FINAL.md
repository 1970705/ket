# Onboarding Design: 保留孩子的前期游戏体验方案

**Document Version**: 2.0
**Created**: 2026-02-17
**Designer**: game-designer
**Priority**: P0 - 用户留存
**Target Audience**: 10岁儿童

---

## 核心原则

**游戏性 > 学习效果** (前1-2周)

> 孩子不会玩让他们感到挫败的游戏。必须先建立情感连接和游戏习惯，再增加学习深度。

---

@Composable
fun ComboIndicator(
    streak: Int,
    modifier: Modifier = Modifier
) {
    // 大号字体，位置显著
    Column(
        horizontalAlignment = Alignment.CenterHorizontally,
        modifier = modifier
            .graphicsLayer {
                // 火焰跳动动画
                scaleX = if (streak >= 3) 1.2f else 1f
                scaleY = if (streak >= 3) 1.2f else 1f
            }
    ) {
        if (streak >= 3) {
            Text(
                text = "🔥",
                style = MaterialTheme.typography.displayLarge
            )
        }
        Text(
            text = "$streak 连击!",
            style = MaterialTheme.typography.headlineMedium,
            color = if (streak >= 5) Color.Red else Color.Orange
        )
    }
}

### 2.2 难度自适应系统

```kotlin
// 难度调整逻辑
data class AdaptiveDifficulty(
    val currentLevel: Int,
    val consecutiveCorrect: Int,
    val consecutiveWrong: Int,
    val avgResponseTime: Long
)

fun adjustDifficulty(state: AdaptiveDifficulty): DifficultyAction {
    return when {
        // 连续答对3题 → 增加难度
        state.consecutiveCorrect >= 3 -> DifficultyAction.INCREASE {
            // 去掉提示
            // 缩短选择时间
            // 增加干扰选项
        }

        // 连续答错2题 → 降低难度
        state.consecutiveWrong >= 2 -> DifficultyAction.DECREASE {
            // 给出首字母提示
            // 减少选项数量
            // 宠物主动鼓励
        }

        // 回答太快（可能猜测）→ 保持难度，不奖励combo
        state.avgResponseTime < 2000 -> DifficultyAction.MAINTAIN_NO_COMBO

        else -> DifficultyAction.MAINTAIN
    }
}

sealed class DifficultyAction {
    data class INCREASE(val action: () -> Unit) : DifficultyAction()
    data class DECREASE(val action: () -> Unit) : DifficultyAction()
    object MAINTAIN_NO_COMBO : DifficultyAction()
    object MAINTAIN : DifficultyAction()
}
```

## 设计决策记录

### 为什么先做选择题？

1. **零挫败感**: 100%成功率，建立信心
2. **快速上手**: 无需学习拼写，直接参与
3. **视觉优先**: 图片比文字更吸引孩子
4. **渐进暴露**: 先熟悉单词，再学拼写

### 为什么用宠物而非头像？

1. **情感连接**: 宠物更可爱，更有陪伴感
2. **反馈载体**: 宠物可以作为情绪表达的载体
3. **非竞争性**: 不会出现"我的头像比你丑"的情况
4. **收集动力**: 解锁新宠物是长期动力

### 为什么社交元素非竞争性？

1. **目标年龄**: 10岁孩子抗压能力有限
2. **教育目标**: 学习不应该有"失败者"
3. **正向激励**: 强调进步而非排名
4. **避免焦虑**: 不让孩子因为排名而焦虑

### 为什么引入Boss Battle？

1. **游戏高潮**: 每个关卡需要高潮体验
2. **挑战感**: 满足喜欢挑战的孩子
3. **视觉冲击**: Boss战是最视觉化的设计
4. **成就感**: 击败Boss的成就感更强
