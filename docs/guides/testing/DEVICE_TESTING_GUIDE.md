### 3.1 截图方法

#### 基本截图命令

```bash
# 方法1: 直接截图到电脑
adb shell screencap -p > screen.png

# 方法2: 截图保存到设备后拉取
adb shell screencap -p /sdcard/screen.png
adb pull /sdcard/screen.png

# 方法3: 截图指定界面（需要先进入界面）
adb shell am start -n com.wordland/.ui.MainActivity
sleep 2  # 等待界面加载
adb shell screencap -p > home_screen.png
```

#### 录屏命令

```bash
# 开始录屏（最长3分钟）
adb shell screenrecord /sdcard/demo.mp4
```

### 3.8 常见视觉问题快速判断

| 症状 | 可能原因 | 快速验证 |
|------|---------|---------|
| 文字被裁剪 | 容器高度太小 | 检查Modifier.height()值 |
| 文字重叠 | Column/Row间距不足 | 检查Arrangement.spacedBy() |
| 按钮文字不完整 | padding不够 | 检查ButtonDefaults.padding |
| 中文显示异常 | 字体不支持 | 换设备测试对比 |

### 3.9 视觉QA验收标准

**阻塞条件**（任一即不通过）:
- ❌ 文字被裁剪导致不可读
- ❌ UI元素超出屏幕边界
- ❌ 按钮内容无法辨认
- ❌ 中文显示乱码
- ❌ 触摸目标被遮挡

**警告条件**:
- ⚠️ 间距轻微不一致（可延后修复）
- ⚠️ 某设备上的对齐偏差
