## 5. 音频资源需求

### 5.1 单词发音 (P1优先级)

**需求量**: 180个单词 (MVP)

**格式要求**:
- 文件格式: MP3
- 比特率: 128 kbps
- 采样率: 44.1 kHz
- 声道: 单声道
- 时长: 每个约1秒
- 总大小: ~1.8 MB

**命名规范**:
```
assets/audio/words/{word_id}.mp3
示例: assets/audio/words/look_001.mp3
```

**录制要求**:
- ✅ 标准英式发音 (RP)
- ✅ 清晰、缓慢、适合儿童
- ✅ 女性声音 (更亲切)
- ✅ 无背景噪音
- ✅ 音量一致

### 5.2 例句音频 (P1优先级)

**需求量**: 200+例句 (MVP)

**格式要求**:
- 文件格式: MP3
- 比特率: 128 kbps
- 采样率: 44.1 kHz
- 时长: 每句约3秒
- 总大小: ~2.4 MB

**命名规范**:
```
assets/audio/sentences/{word_id}_{index}.mp3
示例: assets/audio/sentences/look_001_0.mp3
```

### 5.3 音效文件 (P2优先级)

**需求列表**:

## 7. 内容生产计划

### 7.1 词汇生产流程

```
1. 选词 → KET词表筛选
2. 数据 → 添加音标、翻译、例句
3. 词根 → 添加root/prefix/suffix
4. 审核 → 教育专家审核
5. 测试 → 单元测试数据验证
6. 集成 → 融入游戏
```

### 7.2 音频生产流程

```
1. 脚本 → 准备发音列表
2. 录制 → 专业配音录制
3. 后期 → 降噪、剪辑、标准化
4. 审核 → 发音准确性检查
5. 集成 → 添加到应用
```

### 7.3 图片生产流程

```
1. 设计 → 确定美术风格
2. 草图 → 每个单词草图
3. 绘制 → 数字绘画
4. 审核 → 风格一致性检查
5. 优化 → 压缩、格式转换
6. 集成 → 添加到应用
```

## 8. 内容质量标准

### 8.1 词汇数据质量

**每个单词必须包含**:
- ✅ 单词拼写 (word)
- ✅ 国际音标 (pronunciation)
- ✅ 中文翻译 (translation)
- ✅ 词性标注 (partOfSpeech)
- ✅ 难度等级 (difficulty: 1-5)
- ✅ KET/PET标记 (ketLevel, petLevel)
- ✅ 例句 (exampleSentences: JSON数组)
- ✅ 相关词 (relatedWords: JSON数组)
- ✅ 词根词缀 (root, prefix, suffix)
- ✅ 主题标签 (theme)

**例句质量要求**:
- ✅ 简单易懂 (适合10岁儿童)
- ✅ 展示单词用法
- ✅ 提供中文翻译
- ✅ 每个单词2个例句

### 8.2 音频质量标准

**技术标准**:
- ✅ 无噪音、无爆音
- ✅ 音量一致 (-16 LUFS)
- ✅ 标准英式发音
- ✅ 清晰、缓慢

**发音标准**:
- ✅ 符合RP发音标准
- ✅ 儿童友好语速
- ✅ 情感适中

### 8.3 图片质量标准

**技术标准**:
- ✅ 分辨率符合要求
- ✅ 文件大小优化
- ✅ 格式正确
- ✅ 透明度处理正确

**设计标准**:
- ✅ 风格统一
- ✅ 色彩鲜艳
