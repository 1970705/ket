## Audio Files (60 MP3s)

### Technical Specifications
- **Format**: MP3
- **Sample Rate**: 44.1 kHz
- **Channels**: Mono (single channel)
- **Bitrate**: 128 kbps (constant)
- **Duration**: 0.5s - 2s per word
- **Volume Normalization**: -16 LUFS (EBU R128 standard)
- **Max Peak**: -1.0 dB (headroom)

### Recording Guidelines
1. **Voice Talent**: Native English speaker (UK preferred for KET)
2. **Speed**: Clear, moderate pace (not too fast for children)
3. **Tone**: Friendly, encouraging, neutral emotion
4. **Pronunciation**: Standard British/International English
5. **Environment**: Professional studio or quiet room with minimal reverb
6. **Post-Processing**:
   - Normalize to -16 LUFS
   - Remove background noise
   - Light compression (2:1 ratio, -20dB threshold)
   - Fade in/out (10ms) to avoid clicks

### File Naming Convention
- Format: `{word}.mp3` (lowercase)
- Example: `look.mp3`, `see.mp3`, `watch.mp3`
- No spaces, hyphens, or special characters
- Place in: `app/src/main/assets/audio/`

## Image Files (14 PNGs)

### Technical Specifications
- **Format**: PNG (supports transparency)
- **Alternative**: JPG for photographs (no transparency needed)
- **Color Depth**: 32-bit ARGB (8888)
- **Compression**: PNG level 9 (max compression, slowest save)
- **JPG Quality**: 85% (good balance)
- **Resolution**:
  - Island cards: 800x600px
  - Level previews: 600x400px
  - Backgrounds: 1080x1920px (full screen)
- **DPI**: 72 DPI (screen standard)

### Art Style Guidelines
1. **Visual Style**:
   - Cartoon/illustration style (child-friendly)
   - Bright, saturated colors
   - Clear outlines and shapes
   - Consistent character design

2. **Island Illustrations**:
   - Distinct themes per island (nature, movement, etc.)
   - Prominent landmarks/features
   - Clear focal point
   - 800x600px, centered composition

3. **Level Previews**:
   - Show learning environment
   - Include relevant visual cues
   - 600x400px, landscape

4. **Backgrounds**:
   - 1080x1920px, portrait
   - Subtle patterns (not distracting)
   - Space for UI overlay
   - Non-repeating seamless patterns

- Islands: `island_{island_id}.png`

- Backgrounds: `bg_{scene_id}.png`

## Notes

### Audio Recording Studio Setup (Recommended)
- **Microphone**: Large-diaphragm condenser (e.g., Audio-Technica AT2020)
- **Interface**: Focusrite Scarlett 2i2 or equivalent
- **Software**: Adobe Audition, Audacity (free), or Pro Tools
- **Environment**: Treated room or portable vocal booth
- **Monitoring**: Closed-back headphones (Audio-Technica M50x)

### Image Creation Tools
- **Vector**: Adobe Illustrator, Inkscape (free)
- **Raster**: Adobe Photoshop, GIMP (free), Procreate (iPad)
- **Export**: PNG with transparency (alpha channel)
- **Compression**: TinyPNG (online tool) for optimization

### Alternative: Stock Assets
If recording/creating from scratch is not feasible:
- **Audio**: Freesound.org (CC0 license), ElevenLabs (AI voices)
- **Images**: Freepik.com (educational license), OpenClipart.org
