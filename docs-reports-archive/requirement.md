## 一、项目概述

### 1.1 项目简介

Wordland 是一款面向 10-12 岁儿童的 KET/PET 词汇学习游戏应用，通过游戏化方式帮助儿童掌握核心英语词汇。

**核心价值**:
- 🎮 **游戏化学习**: 让背单词像玩游戏一样有趣
- 🧠 **科学记忆**: 基于艾宾浩斯遗忘曲线的智能复习系统
- 📊 **可视化进度**: 家长可以清晰看到孩子的学习成果
- 🎯 **考试提分**: 专注 KET/PET 考试核心词汇

### 1.2 技术栈

- **平台**: Android (原生)
- **语言**: Kotlin
- **UI 框架**: Jetpack Compose
- **架构**: Clean Architecture
- **数据库**: Room (SQLite)
- **依赖注入**: Hilt
- **异步**: Coroutines + Flow

### 1.3 目标用户

- **主要用户**: 10-12 岁儿童（小学 4-6 年级）
- **次要用户**: 家长（查看学习报告、设置学习目标）
- **目标市场**: 准备 KET/PET 考试的中国学生

## 四、功能需求详细说明

### 4.1 用户流程

#### 新用户流程
```
1. 启动应用
2. 查看 HomeScreen（欢迎界面）
3. 点击"开始学习" → 进入 IslandMap
4. 选择 Look Island → 进入 LevelSelect
5. 选择 Level 1 → 开始学习
6. 完成 6 个词 → 关卡完成
7. 返回关卡选择 → Level 2 解锁
8. 继续学习...
```

#### 复习流程
```
1. 在 HomeScreen 点击"每日复习"
2. 查看待复习词汇列表
3. 选择一个词 → 开始复习
4. 完成复习 → 更新记忆强度
5. 继续下一个复习词...
```

#### 家长查看流程
```
1. 在 HomeScreen 切换到家长模式
2. 查看 ProgressScreen
3. 选择时间范围（本周/本月）
4. 查看学习报告
5. 导出报告（可选）
```

### 4.2 游戏机制

#### 拼写战斗（已实现）
- **输入方式**: 虚拟 QWERTY 键盘
- **提示**: 显示首字母
- **反馈**: 实时颜色（蓝色正确，红色错误）
- **评分**: 基于准确率、速度、提示使用

#### 听音寻宝（待实现）
- **输入方式**: 点击图片选项
- **音频**: 播放单词发音
- **提示**: 再次播放音频
- **反馈**: 正确/错误动画

#### 句子配对（待实现）
- **输入方式**: 拖拽单词到空位
- **提示**: 显示首字母
- **反馈**: 语法检查，正确/错误提示

#### 快速判断（待实现）
- **输入方式**: 点击正确/错误按钮
- **时间限制**: 10 秒倒计时
- **反馈**: 即时对错 + 连击奖励

### 4.3 核心算法

#### 记忆强度算法（已实现）
```kotlin
初始强度: 10
正确加成: +10
错误惩罚: -15
蒙答惩罚: -20 (蒙对), -20 (蒙错)

下次复习时间 = 当前时间 + (强度 / 10) 天
```

#### 星星评分算法（待实现）
```kotlin
基础分数 = 正确率 × 100

速度加成:
  - 平均响应时间 < 3 秒: +10 分
  - 平均响应时间 < 5 秒: +5 分
  - 平均响应时间 >= 5 秒: 0 分

提示惩罚:
  - 使用 1 次提示: -5 分
  - 使用 2 次提示: -10 分
  - 使用 3+ 次提示: -15 分

星星评定:
  - >= 90 分: ⭐⭐⭐ (3 星)
  - >= 70 分: ⭐⭐ (2 星)
  - >= 50 分: ⭐ (1 星)
  - < 50 分: ❌ (需重试)
```

#### 精通度判定（已实现）
```kotlin
精通条件:
  1. 连续 3 次正确
  2. 记忆强度 >= 60
  3. 跨场景正确率 >= 70%
```
